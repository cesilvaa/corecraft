# CoreCraft

Repositorio de estudos e atividades do programa CoreCraft, da BitcoinCoders.

O objetivo deste repositório é organizar os trabalhos desenvolvidos ao longo do curso, com foco em entender o Bitcoin Core em nível de arquitetura e operá-lo de ponta a ponta: linha de comando, integração via RPC e ZMQ, interpretação da mempool, acompanhamento de transações e reação a eventos da rede em tempo real.

## Estrutura

```text
corecraft/
├── atividade-1/
├── atividade-2/
├── atividade-3/
└── README.md
```

## Atividades

- `atividade-1/`: primeira atividade do curso.
- `atividade-2/`: segunda atividade do curso.
- `atividade-3/`: terceira atividade do curso.

Cada atividade deve conter seu próprio README, scripts, configurações e anotações conforme os trabalhos forem desenvolvidos.

## Requisitos esperados

Ao longo do curso, este repositório pode incluir exemplos e integrações usando:

- Bitcoin Core
- `bitcoin-cli`
- RPC
- ZMQ
- scripts de automação
- aplicações para monitorar mempool, transações e eventos da rede
